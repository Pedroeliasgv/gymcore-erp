# 💪 Ugym ERP

Sistema web de gestão para academias.

## 🚀 Tecnologias

- Frontend: HTML, CSS, JavaScript
- Backend: C# (.NET Web API)
- Banco: SQLite
- ORM: Entity Framework Core

## 📦 Funcionalidades

- Gestão de alunos
- Controle financeiro
- Planos e contratos
- Check-in
- Dashboard com indicadores
