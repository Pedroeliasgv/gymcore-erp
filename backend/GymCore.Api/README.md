## 🚀 Sobre o projeto

O **GymCore ERP** é um sistema web desenvolvido para resolver a desorganização operacional de academias, centralizando todas as áreas do negócio em uma única plataforma.

> ⚠️ Sistema voltado para uso interno (administração, recepção, gestores e instrutores)

---

## 🎯 Problema

Academias pequenas e médias enfrentam:

- ❌ Controle financeiro em planilhas
- ❌ Cadastro de alunos desorganizado
- ❌ Falta de controle de inadimplência
- ❌ Decisões sem base em dados

---

## 💡 Solução

O GymCore resolve isso com um sistema integrado contendo:

- 👥 Gestão de alunos
- 💰 Controle financeiro
- 📦 Planos e contratos
- 🏋️ Gestão de treinos
- 📊 Dashboard e relatórios
- ⚙️ Controle operacional
